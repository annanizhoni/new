Task 1: Test Flashcard and Quiz Features

Please complete each of the following steps:
  1. Select a course of your choice.
  2. Select the flashcard button.
  3. Tap the flashcard.
  4. Return to the course you selected.
  5. Tap the Quiz button and take the one question quiz.
  6. If you selected the correct answer, tap the redeem button to visit the shop
  7. If you selected the incorrect answer, please retake the quiz until you do.
